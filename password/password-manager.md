## What Is a Password Manager?

A **Password Manager** is an app that **securely stores all your passwords in one safe place**.

Instead of remembering dozens of passwords, you only remember **one strong master password**, and the password manager does the rest.

It can:
- Save your passwords securely
- Create strong passwords for you
- Automatically fill login details on websites and apps
- Keep your data encrypted (locked) so no one else can read it

Think of it as a **digital locker** for your passwords.

---

## Types of Password Managers

### 1. Cloud-Based Password Managers
- Work across multiple devices
- Sync passwords automatically
- Best for most users

Example: Using the same passwords on phone and laptop without effort

### 2. Offline / Local Password Managers
- Passwords stored only on your device
- No cloud syncing unless you do it manually
- Better for advanced or privacy-focused users

### 3. Browser-Based Password Managers
- Built into browsers like Chrome, Edge, Safari
- Easy but less powerful than dedicated apps

## Tips & Tricks for Using a Password Manager Safely

- Use a **strong master password** that you can remember but others cannot guess
- Never share your master password with anyone
- Enable **two-factor authentication (2FA)** if available
- Let the password manager generate passwords for you
- Do not write passwords in notes or screenshots
